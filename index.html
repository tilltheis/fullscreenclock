Fullscreen Clock

Display a clock on all of your inactive screens when running an application in fullscreen mode.

Screenshot

Appearance

You can change the appearance settings via the status bar icon or by double-clicking fullscreenclock.app when it's already running.

System Requirements

You need at least Mac OS X 10.6 Snow Leopard.

Running Under Mac OS X 10.9 Mavericks

When running the app under Mavericks for the first time you may get a prompt that says that the app is not downloaded from the App store or that it is from an unidentified developer. That is because I don't have an official Mac developer ID which costs money and I don't make any money from my apps. You can install the app anyway by following the instructions on this page.
